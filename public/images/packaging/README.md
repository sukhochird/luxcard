# Бэлэглэлийн бүтээгдэхүүний зургууд

Энэ хавтсанд бодит бүтээгдэхүүний зургуудыг нэмнэ. Файлын нэрс нь `data/packaging.json` дахь `id`-тай таарна:

- `standard.jpg` — Стандарт дугтуй
- `premium.jpg` — Дээд зэргийн хайрцаг
- `giftbag.jpg` — Бэлгийн уут
- `box.jpg` — Гадны дугтуй
- `flowers.jpg` — Цэцгийн баглаа
- `chocolate.jpg` — Шоколад
- `candle.jpg` — Гэрэлт тахиа
- `teddy.jpg` — Дүүгэлзэг
- `card_holder.jpg` — Картны хайрцаг
- `ribbon.jpg` — Дээд зэргийн боолт
- `basket.jpg` — Бэлгийн саг
- `wine.jpg` — Дарс
- `balloon.jpg` — Бөмбөлөг

Зураг байхгүй тохиолдолд дэлгэцэн дээр default дүрс гарна.
